Vacation Manager App

The Vacation Manager is an Android mobile application designed to help users plan, organize, and manage their vacations and associated excursions. The application allows users to create vacation itineraries with alerts and sharing capabilities. The app securely stores user data in a local SQLite database.

App Purpose

This application addresses the need for a comprehensive vacation planning tool that allows users to:

  • Organize multiple vacations
  • Plan multiple excursions for each vacation
  • Receive notifications on vacation and excursion dates
  • Share vacation details via email, SMS, and clipboard
  • Manage all data locally without requiring an internet connection

System Requirements

  • Minimum Android Version: Android 8.0 (API Level 26)
  • Target Android Version: Android 13 or newer (API Level 36)
